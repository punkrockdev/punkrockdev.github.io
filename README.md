# Punk Rock Dev website

Install Ruby, node.js (optional) and prepare your local environment:

    bundle install
    npm install
    npm install -g gulp-cli

Then develop using:

    jekyll serve

Deploy by commiting, but first optimize images:

    gulp

Live on: <http://punkrockdev.com/>
